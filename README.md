# OnlyofficeLanguageHelper

ONLYOFFICE Language Helper. Used in QA

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'onlyoffice_language_helper'
```

And then execute:

```shell script
bundle
```

Or install it yourself as:

```shell script
gem install onlyoffice_language_helper
```
